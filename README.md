# vistoria app

## backend
usar branch main
cd  vistoria-app/backend

### instalar dependencias
npm i

### rodar servidor backend em
node server.js

## frontend
usar a branch frontend
cd vistoria-app/front-end

### instalar dependencias
npm i

### rodar react localmente

npm start
